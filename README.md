# 🎮 Jogo da Velha com IA 🤖

Um clássico **Jogo da Velha** desenvolvido com inteligência artificial para desafiar seus limites!  
O jogador poderá enfrentar outro humano ou competir contra uma IA estratégica, criada para nunca perder.  
Prepare-se para testar suas habilidades e tentar encontrar a falha na máquina! 😏

---

## 📌 Funcionalidades

✅ Modo **Jogador vs Jogador** (PVP)  
✅ Modo **Jogador vs IA** (PVE)  
✅ Interface simples e intuitiva  
✅ IA baseada em **algoritmo Minimax** para decisões estratégicas  
✅ Detecção automática de vitória, empate ou jogo em andamento  
✅ Reinício rápido da partida
